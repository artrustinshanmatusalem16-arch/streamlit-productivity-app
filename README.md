# Student Productivity Tracker (Streamlit App)

## Description
This Streamlit application helps students track their study habits, focus level, and productivity.

## Features
- Study session tracker
- Productivity dashboard
- Charts and metrics
- Interactive UI with Streamlit components

## Target Users
High school and college students who want to monitor their study productivity.

## Technologies Used
- Python
- Streamlit
- Pandas

## How to Run
Install dependencies:

pip install -r requirements.txt

Run the app:

streamlit run app.py
